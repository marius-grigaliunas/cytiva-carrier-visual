# Carrier dashboard

This is a test of a dashboard for cytiva.

## Loading the report

- **From server:** Copy your Excel report to `public/data/report.xlsx`, then click “Load report from server”.
- **From file:** Click “Choose Excel file…” and select your report (e.g. from the `data/` folder).

Deliveries are grouped by carrier using the ship method column (DHL_, DHLFRT_, EXPEDITORS_, KN_AIR, GEODIS_, and Other).
